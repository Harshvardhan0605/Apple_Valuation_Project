Project Overview: Apple Inc. Intrinsic Valuation & Sensitivity Hub
The Objective
This project provides a comprehensive valuation of Apple Inc. (AAPL), bridging the gap between traditional 3-statement financial modeling and modern data visualization. As an aspiring Financial Analyst, my goal was to move beyond static spreadsheets and build a dynamic tool that allows stakeholders to visualize how shifting macroeconomic assumptions—like WACC and Terminal Growth—impact equity value in real-time.
Financial Modeling Methodology
The foundation of this project is a bottom-up, integrated 3-statement model (Income Statement, Balance Sheet, and Cash Flow Statement) for a 5-year forecast period (2026–2030).
DCF Triangulation: I utilized a dual-method Discounted Cash Flow (DCF) approach, calculating intrinsic value via the Perpetuity Growth Method (assuming a 2.5% $g$) and the Exit Multiple Method (utilizing a 22.0x EBITDA multiple).
Cost of Capital: A WACC of 8.6% was derived using the Capital Asset Pricing Model (CAPM), reflecting a conservative risk profile for the current market environment.
The Investment Thesis: A 33.9% Downside
My analysis indicates that Apple is currently trading at a significant premium to its fundamental value. By blending both DCF methods, I arrived at a target price that represents a 33.9% downside from the current market price of $258.87. While Apple remains a cash-flow powerhouse, the current market valuation requires aggressive growth assumptions that significantly exceed conservative long-term projections.
Technical Innovation: The Excel-to-Power BI Bridge
To enhance usability, I developed a navigation hub within Excel using a Relative Path formula. This creates a seamless "one-click" bridge that launches a local Power BI Sensitivity Dashboard.
Power BI Integration: The dashboard uses DAX measures to visualize thousands of valuation scenarios, allowing users to stress-test the model against fluctuating interest rates and growth targets.
Portfolio Ready: By utilizing relative paths, the project remains fully portable; once downloaded, the interactive link recalibrates to the user's local directory automatically.
